| **类别**   | **字段名 (建议)**                          | **作用**                            |
| ---------- | ------------------------------------------ | ----------------------------------- |
| **基础**   | `method`, `url`, `status_code`             | 业务逻辑核心                        |
| **网络**   | `src_ip`, `src_port`, `dst_ip`, `dst_port` | 确定通信双方                        |
| **性能**   | `rtt` (往返时间), `duration`               | 分析接口慢不慢                      |
| **安全**   | `user_agent`, `ja3`, `referer`             | 溯源与反爬虫                        |
| **元数据** | `sensor_id`, `interface`                   | 确定流量是从哪台机器/哪个网卡抓到的 |
